Sorteador de Letras - Jogo Stop

Este projeto foi desenvolvido com o objetivo de praticar lógica de programação e construção de interfaces web utilizando HTML, CSS e JavaScript.

A proposta é simples: gerar uma letra aleatória do alfabeto para ser utilizada em jogos como Stop (Adedonha). A cada clique, o sistema sorteia uma nova letra, sem repetições, até que todas sejam utilizadas.

Tecnologias utilizadas
- HTML
- CSS
- JavaScript

Funcionalidades
- Sorteio aleatório de letras (A a Z)
- Não repete as letras sorteadas
- Botão para reiniciar o jogo
- Interface simples e intuitiva

Objetivo do projeto
Este projeto foi inspirado em exercícios de lógica de programação, mas adaptado para uma proposta própria, com foco na prática e consolidação dos conceitos aprendidos.

Como usar
1. Clique no botão "Sortear"
2. Uma letra será exibida na tela
3. Continue até completar todas as letras
4. Use "Reiniciar" para começar novamente
